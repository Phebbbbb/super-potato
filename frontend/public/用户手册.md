## 1. 快速开始

### 登录系统

1. 打开浏览器访问系统地址
2. 输入用户名和密码
3. 默认管理员账号：`admin` / `admin123`

### 切换客户

顶部导航栏右侧有客户选择器，点击下拉切换不同客户，所有页面数据自动隔离。

### 菜单导航

左侧边栏按代账业务流程组织：

| 菜单分组 | 功能 |
|---------|------|
| 代账业务 | 客户管理、票据中心、记账中心、内审中心、报表中心、报税中心 |
| 增值服务 | 薪酬管理、银企对账、外勤任务 |
| 智能工具 | RPA 调度、QR 追溯、税务风控、AI 税务顾问 |

## 3. 客户管理

管理代账公司服务的所有企业客户。

### 新增客户

1. 点击「新增客户」按钮
2. 填写客户名称、税号、纳税人性质（一般/小规模）、行业、联系人
3. 点击保存

### 编辑/删除

点击操作列的编辑或删除按钮。**仅管理员可操作**。

## 4. 票据中心

所有原始凭证（发票、银行回单、合同等）的统一管理入口。

### 手动上传

1. 点击「上传票据」
2. 选择文件，填写票据类型
3. 系统自动 OCR 识别

### RPA 自动采集

通过 RPA 调度模块，影刀/来也等 RPA 工具可自动推送扫描件到系统。详见 [RPA 调度](#13-rpa-调度)。

### 查看 OCR 结果

点击票据旁的「OCR」标签，查看识别出的结构化数据（发票号码、金额、买卖方等）。

## 5. 记账中心

核心模块。从票据生成记账凭证，支持 AI 自动生成和手工录入。

### AI 智能生成凭证

1. 点击「AI 生成凭证」
2. 勾选已 OCR 完成的票据（可多选）
3. 点击确认，AI 自动生成借贷分录
4. 凭证进入「草稿」状态，待审核

### 手工录入

1. 点击「手工录入」
2. 填写凭证摘要、日期
3. 逐行添加科目编码、科目名称、借方/贷方金额
4. 系统自动校验借贷平衡

### 审核确认

- 点击「审核」→ 填写审核人姓名 → 确认通过
- 确认后生成 QR 追溯码，**不可再修改**

### 驳回

- 点击「驳回」→ 填写驳回原因 → 凭证状态变为「已驳回」
- 可重新编辑修正后再次提交审核

### 审计日志

点击「日志」查看凭证的完整操作记录（创建、修正、审核、驳回）。

### 批量操作

- 勾选多张凭证
- 点击「批量确认」一键审核多张

## 6. 内审中心

独立的内审工作台，用于复核 AI 生成的凭证和申报数据。

### 功能

- **凭证审计队列**：待审核的凭证列表，支持审核通过（含质量评分）和驳回
- **申报审计队列**：待审核的纳税申报
- **内审记录**：所有审计操作日志
- **导出报告**：支持导出 CSV 格式内审报告

### 质量评分

审核凭证时可给 1-5 星评分：
- 1-2 星：需改进
- 3 星：合格
- 4 星：良好
- 5 星：优秀

## 7. 报表中心

自动从已确认凭证生成财务报表。

### 支持的报表

| 报表 | 说明 |
|------|------|
| 科目余额表 | 各科目期初/本期/期末借贷余额 |
| 利润表 | 营业收入 - 成本 - 费用 = 净利润 |
| 资产负债表 | 资产 = 负债 + 所有者权益（自动平衡） |
| 总账 | 按日期排序的全部凭证分录 |

### 操作

- 顶部选择会计期间（月份）
- 点击「导出」下载 CSV 格式报表

## 8. 报税中心

纳税申报任务管理与自动申报。

### 创建申报

1. 点击「创建申报」
2. 选择税种（增值税/企业所得税/附加税）和所属期
3. 点击「预览申报数据」→ 系统自动从凭证汇总计算
4. 确认数据无误后点击「确认创建」

### 申报数据预览

| 税种 | 自动计算项 |
|------|-----------|
| 增值税（小规模） | 销售额 × 3%，月≤10万自动减免 |
| 增值税（一般） | 销项税额 - 进项税额 |
| 企业所得税 | 利润 × 25% |
| 附加税 | 城建税 7% + 教育附加 3% + 地方教育附加 2% |

### 审核申报

- 点击「审核」→ 选择通过/驳回 → 填写审核意见

### 自动申报（Playwright 引擎）

1. 在系统设置中配置电子税务局登录凭据
2. 点击「自动申报」
3. 系统自动打开浏览器 → 登录电子税务局 → 填写申报表 → 提交 → 截图留证

> **注意**：首次使用需在服务器安装 `playwright install chromium`。建议先用 headless=false 模式测试。

## 9. 全自动加工链

一键完成"票据 → 凭证 → 确认 → 申报"的全流程自动化。

### 触发方式

**方式一：RPA 调度页面**

1. 进入「智能工具 → RPA 调度」
2. 点击「全自动加工」按钮
3. 系统自动处理所有未使用的 OCR 票据

**方式二：API 调用**

```
POST /api/rpa/auto-process?client_id=c001
```

### 处理逻辑

```
OCR 完成的票据
    │
    ▼
按日期分组（同日 2+ 张合并一张凭证）
    │
    ▼
AI 规则引擎生成借贷分录
    │
    ▼
借贷平衡校验 + 自动修正
    │
    ▼
判断置信度
    ├── 高置信度（≤3 行 + 标准科目）→ 自动确认
    └── 低置信度 → 草稿状态，入复核队列
    │
    ▼
自动为涉及月份创建增值税申报任务
```

## 12. 外勤任务

线下跑腿任务纳入系统追踪。

### 任务类型

| 类型 | 说明 |
|------|------|
| tax_bureau | 税务局办理 |
| business_reg | 工商注册 |
| bank | 银行事务 |
| client_visit | 客户拜访 |
| document_delivery | 文件递送 |
| other | 其他 |

### 任务流程

创建 → 分派给外勤人员 → 执行中 → 完成后上传现场照片 → 完成

### 优先级

- 紧急（urgent）：红色标记
- 高（high）：橙色
- 普通：默认

## 13. RPA 调度

管理 RPA 任务的创建、下发和触发。

### 任务类型

- **scan_invoice**：发票扫描采集
- **file_tax**：纳税申报
- **data_sync**：数据同步

### 操作

- **创建任务**：手动创建 RPA 任务并触发
- **一键触发全部**：触发所有待处理任务
- **全自动加工**：一键完成全流程自动化（推荐）

### RPA 对接

系统提供标准 API 供第三方 RPA 工具调用：

```
POST /api/rpa/ingest    — RPA 推送票据数据
GET  /api/rpa/tasks     — 查询待处理任务
POST /api/rpa/trigger/{id} — 触发单个任务
```

## 14. QR 追溯

全流程溯源追踪。

### 查询方式

1. 输入凭证编号（如 `JZ-202606-0001`）
2. 输入 QR 码 ID
3. 输入 `类型:ID` 格式（如 `voucher:xxx-xxx`）

### 追溯节点

| 节点 | 颜色 | 含义 |
|------|------|------|
| 票据入库 | 蓝色 | 原始凭证进入系统 |
| AI 记账 | 紫色 | AI 生成凭证 |
| 审核确认 | 绿色 | 人工审核通过 |
| 纳税申报 | 橙色 | 提交税务申报 |
| 报表生成 | 青色 | 财务报表生成 |

每条链记录完整的时间点和操作人员，支持向后追溯到原始票据，向前追溯到申报结果。

## 15. 税务风控

### 申报日历

展示未来 3 个月各税种申报截止日期：
- **急迫**（7 天内）：橙色标记
- **逾期**：红色标记

### 风险检测

自动检测以下风险项：
- 税负率异常
- 进销项不匹配
- 长期零申报
- 发票缺失

风险评分：
- **A 级**：健康（绿色）
- **B 级**：关注（橙色）
- **C 级**：危险（红色）

## 18. 常见问题

### Q: 凭证确认后还能修改吗？
A: 不能。已确认凭证不可修改，这是审计合规要求。需要修正请联系管理员走驳回流程。

### Q: 为什么利润表显示负数？
A: 这可能是因为当月只有费用没有收入，或者期初余额未录入。正常运行后数据会正常。

### Q: 资产负债表为什么不平衡？
A: 系统已自动将本期损益计入未分配利润，确保 资产 = 负债 + 所有者权益。如果不平衡，请检查是否有未确认的凭证。

### Q: 如何添加新的省份电子税务局配置？
A: 编辑 `backend/app/services/tax_automation.py`，在 `TAX_BUREAU_CONFIGS` 中添加新的省份配置（登录 URL 和页面选择器）。

### Q: Playwright 申报引擎需要什么前置条件？
A: 服务器安装 `pip install playwright && playwright install chromium`。建议先用 headless=false 模式调试选择器是否正确。

### Q: 普通会计能删除凭证吗？
A: 不能。修改和删除操作需要管理员或审核员角色。普通会计只能录入和查看。

### Q: 如何新增客户？
A: 管理员登录后进入「客户管理」页面，点击新增。客户数据完全隔离，切换客户后各页面数据自动切换。
